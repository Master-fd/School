#校园社团App源码
##App目的  

**专注于校园社团和学生之间的App，解决每次社团招新的时候，需要大批量的派送传单进行宣传。通过校园社团App，学生关注社团之后，即可查看社团发布的多有相关职位信息，直接发送简历应聘，提高应聘效率**  
##App功能  

1、注册用户，分为组织和学生两种。登录时自动识别。  
2、学生用户和组织用户都可以通过二维码或者账号输入的方式互相添加好友，彼此好友之后即可发送消息。  
3、学生用户可以查看关注的组织招聘信息，发送简历应聘。  
4、组织用户可以发布招聘信息，接收简历。
